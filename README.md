# MAT-325
Calc 3, Multivariable Calculus: SNHU 2025

MATLAB assignments and projects submitted as PDF files. All computations and visualizations were produced using MATLAB.

Module 1:
Defining and plotting 2D and 3D vectors with quiver() / quiver3(),
Dot products and cross products,
Plotting parametric lines with fplot3() and planes with fsurf(),
Quadratic surfaces using meshgrid() and surf()

Module 2:
Plotting vector-valued functions (parametric curves in 3D),
Computing and plotting the unit tangent vector T(t),
Arc length using symbolic integration with int() and vpa()

Module 3:
Plotting multi-variable surfaces with fsurf(),
Evaluating limits of multi-variable functions by visual inspection and algebraic paths,
Computing partial derivatives symbolically with diff()

Project 1:
Normal vectors to planes; plotting with surfnorm(),
Position, velocity, and acceleration of a parametric projectile; velocity vectors plotted with quiver3(),
Unit normal vector N(t) and dot product v(t)·N(t),
Tangent plane approximation plotted alongside the original surface,
Finding and classifying critical points of a multivariable function using the second derivative test

Module 5:
Double integrals over rectangular and non-rectangular regions using integral2(),
Double integrals in polar coordinates (change of variables),
Triple integrals using integral3(); solid regions visualized with trisurf() and convhull(),
Coordinate conversions: rectangular, cylindrical, and spherical

Module 6:
Plotting vector fields with quiver() (scaled and unscaled),
Evaluating line integrals using integral(),
Green's Theorem: converting line integrals to double integrals with integral2()

Project 2:
Volume of solids defined by iterated integrals in cylindrical and spherical coordinates,
Conversion between coordinate systems and verification with integral3(),
Line integrals and work computed over parameterized curves,
Green's Theorem applied to line integrals over closed curves,
Stokes' Theorem: evaluating surface integrals via curl,
Divergence Theorem: converting surface integrals to volume integrals

Reflection:
MAT 325 extended single-variable calculus into three dimensions, which required building geometric intuition alongside algebraic technique. 
Plotting every object in MATLAB — planes, surfaces, vector fields, solid regions — made abstractions concrete. 
Seeing the tangent plane touch a surface at exactly one point, or watching velocity vectors spin around a parametric curve,
connected the formulas to physical meaning in a way that calculation alone cannot.
The most conceptually significant topic was the trio of theorems at the end of the course:Green's, Stokes', and the Divergence Theorem. 
All three say the same thing at different scales — the behavior of a field on a boundary determines what happens in the interior — 
but applying each one requires correctly identifying the geometry and setting up the right coordinate system.
